PalmPad
A single handed keyboard based on the hackclub hackpad. Uses a resistor ladder and custom qmk firmware to allow for more inputs than the seeduino xiao has room for.
This is still a work in progress and the case is just temporary. The final version will look similar to a joystick. It is however difficult to design the joystick without the pcb as it has to be broken into parts. This document will be updated later on.

Prototype of the palmpad, just used to test electronics and clearances.

*Seeeduino xiao rp2040
*16x SK6812 MINI-E LEDs
*2x EC11 Rotary encoders
*14x MX-Style switches
*10x through-hole 1N4148 Diodes
*1x 0.91 inch OLED display
*14x white blank DSA keycaps
*4x M3x16mm screws
